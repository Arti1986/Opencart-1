#Opencart_2.3.x

Плагин: InterKassa Gateway
Описание: Платежный шлюз "Интеркасса" для сайтов Open Cart 2.3.x. (версия протокола Интеркассы 2.0)
Предназначение: Модуль добавляет способ оплаты "Интеркасса" для Open Cart 2.3.x
Версия: 1.02 

#Установка
1. Скачать плагин к себе на компьютер, распаковать
2. Закачать все из папки upload на сервер
3. зайти в админку, найти в списке метод оплаты "InterKassa" и нажать "Активировать"
4. Перейти на вкладку настроек, заполнить все поля, уникальный номер кошелька, секртеный ключ, тестовый ключ при необходимости.

Жмем "Сохранить"

Метод активен и появится в списке оплат вашего магазина.
